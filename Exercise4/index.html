Microsoft SQL Server

What is Microsoft SQL Server?

Microsoft SQL Server is a relational DBMS(Databse Management System). Realeased by Microsoft in the year 1989 it has made its way on the top of DBMS's. What is a DBMS though? A DBMS is a system that gives us ways to manage and work with the incoming data that are stored in a database, in efficient ways. It is a system very useful in the industry of Information Techonolgy since it offers a variety off applications. Those applications include transaction processing, business inteligence and analytics. The system comes with the right tools in order to enable the above mentioned applications.

What is SQL?

Microsoft SQL Server is built on top of SQL(Structured Query Language). SQL allows us to interact with the databases. It is the standard language that is used to store the incoming data, manipulate and retrieve the already existing data. There are many versions of SQL. Microsoft has come up with T-SQL(Transact SQL), making SQL a subset of it. T-SQL comes with more features and more functionalities than the ones offered in the standard SQL.

Editions

Microsoft SQL Server comes in different editions to fit the needs of different individuals and organizations.

Express
Is the best edition for someone new to building small data-driven applications. It is a free edition and has a capacity of 10GB storage.
Developer
Free edition that comea with all the features Microsoft SQL Server has to offer. Its use though is limited to development and testing, not production.
Enterprise
Comes with every feature. It's the premium paid version, ideal for medium and large enterprises.
Standard
It is a paid version ,offering the most standard features, but not all of them. It is ideal for small businesses
Web
It is a paid version ideal for web hosts.
