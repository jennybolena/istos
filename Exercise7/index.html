Microsoft SQL Server

What is Microsoft SQL Server?

Microsoft SQL Server is a relational DBMS(Databse Management System). Realeased by Microsoft and Sybase in the year 1989 it has made its way on the top of DBMS's. What is a DBMS though? A DBMS is a system that gives us ways to manage and work with the incoming data that are stored in a database, in efficient ways. The data is stored in tables, which are considered as database objects. The fundamental logical structure of the tables is in rows(each single entry) and in columns(the attributes of the entry). Microsoft SQL Server is a system very useful in the industry of Information Techonolgy since it offers a variety off applications. Those applications include transaction processing, business inteligence and analytics. A plethora of transactions are handled through it on daily basis. The system comes with the right tools in order to enable the above mentioned applications. It offers high scalability and it can be used from small to large applications and platforms. Everyone can take advantage and use it, since beside the paid versions there are free ones too. It is supported by the operating systems of Microsoft and Linux. It is built on top of SQL(Structured Query Language)

What is SQL?

SQL is the language that is used to communicate and interact with the database. We write SQL queries in order to manipulate the relationships in the database data. Those manipulations include insetion, deletion and update of data. It is an easy to learn language and it is made in such a way that it offers fast data manipulation. It is a powerful tool and knowledge to have considering that we live in a data driven world. There are many versions of SQL. Microsoft has come up with T-SQL(Transact SQL), making SQL a subset of it. T-SQL comes with more features and more functionalities than the ones offered in the standard SQL. With T-SQL we are able to write blocks of code that deifnes procedures. It is a procedural language that allows definition of variables. It also inculdes additional functions for processing strings and dates.

Editions

Microsoft has done very well with Microsoft SQL Server. It offers different versions based on the needs of the ones using it.

Express
Is the best edition for someone new to building small data-driven applications. It is a free edition and has a capacity of 10GB storage.
Developer
Free edition that comes with all the features Microsoft SQL Server has to offer. Its use though is limited to development and testing, not production.
Enterprise
Comes with every feature. It's the premium paid version, ideal for medium and large enterprises.
Standard
It is a paid version ,offering the most standard features, but not all of them. It is ideal for small businesses
Web
It is a paid version ideal for web hosts.
